Mutation

Mutation requires that a single residue is selected. A window appears that allows to mutate this residue to any other natural amino acid. If you choose the original type of amino acid, the sidechain is repacked. The function requires that you have the third-party software SCWRL4 installed and that all backbone atoms of the selected residue are specified in the structure.
