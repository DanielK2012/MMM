What can you do in the site scan setup for nucleotides?

This window appears when a spin labeling site scan is initiated from the EPR/Site scan menu item and the selection or current chain contain only nucleotide residues. Here you can select which types of residues are included in the site scan and you can determine what analysis of the set of labeled residues should be performed.

Residue types

This allows to restrict a sites scan to certain types of residues. Only those residue types can be selected for which rotamer libraries are registered with MMM. The first such residue type is automatically enabled. Please note that there may be restrictions on combining residue types, depending on library. If you are not sure, select only one nucleotide type per site scan.

Membrane depth analysis

In a membrane protein, you might want to know where spin labels are situated with respect to the lipid bilayer. If you have created a lipid bilayer the z axis is the bilayer normal and z=0 corresponds to the central plane of the bilayer. You can analyze all spin-labelled residues for their mean z coordinate and distribution width of the z coordinate by activating the z axis is unique axis checkbox.

Distance analysis

If the scan contains at least two residues, intraspin distances can be analyzed. For large site scans, the computation time and result file size of a complete distance analysis can be rather larger. Therefore it is recommended to restrict the analysis to the information that you really need.

Distances between residues within the same chain are intrachain distances. You can include or exclude them from analysis, by activating radiobutton all or none, respecively.

Distances between residues in different chains are interchain distances. You can include all interchain distances, only distances between residues with the same number in different chains, or exclude all of them from analysis, by activating radiobuttons all, equivalent, or none, respecively.

In a homooligomer (with perfect symmetry), analysis can be sped up by performing the site scan for only a single chain and creating spin label positions in other monomers by symmetry. This assumes that the z axis is the symmetry axis. The multiplicity of the axis (2 for homodimer, 3 for trimer, ...) must be specified.

Additional statistics files

By default, analysis output is written to a HTML file and displayed in a web browser. If you want to track a problem or use quantitative information in other analysis programs, clear text output may be preferable.

To generate text files with full statistics output, activate the checkbox Save statistics. To save separate PDB files for all rotamers with significant population, activate the checkbox Save PDB rotamers. Be careful in both cases, but in particular with the saving of PDB files. In a large site scan this can easily create thousands or even tens of thousands of PDB files.
